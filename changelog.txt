Version: 0.1.7
Date: 2019-05-21

  Changes:
    - Added network ID column to station tab, instead of displaying it as part of control signals (#27).
  Bugfixes:
    - Fixed trains being send to wrong depot when using train info panel (#28).

    - Fixed that no time duration was selected when opening the stats tab for the first time.
